Acessar o site

Ótimo site para pegar ícones de maneira fácil e rápida. Embora não tenha tantas opções disponíveis, o site tem os ícones essenciais, e também logos de empresas/sites/redes sociais.

Como usar

Primeiramente, você deve implementar esse script no final de sua página para começar a usar os ícones.

Você pode encontrar este script na aba "Usage" junto com todas as instruções.

Adicionando ícones no site

Após escolher algum ícone, como por exemplo uma seta para a esquerda, você verá um comando aparecer na parte inferior da tela:

Basta clicar para copiar o código

Agora você deve colar o comando no lugar que deseja no site, por exemplo ao lado de um texto:

Para que o ícone fique do lado do texto, o ícone está posicionado dentro da tag de texto (p) e do lado da palavra "texto"
Se o ícone estivesse posicionado antes do texto, ela apareceria no lado esquerdo da palavra "texto"
